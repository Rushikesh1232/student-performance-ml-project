# 🎓 Student Performance Prediction ML Project

## 📌 Project Overview

This project predicts a student's exam score based on various factors such as:

* Hours Studied
* Attendance
* Sleep Hours
* Previous Score
* Internet Usage

A Machine Learning model was trained using **Scikit-Learn** and deployed as a **web application using Flask**.

---

## 🚀 Live Web Application

You can try the live project here:

🔗 https://student-performance-ml-project-1.onrender.com

The application is deployed on **Render Cloud Platform**.

---

## 🧠 Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training (Scikit-Learn)
5. Model Saving using Joblib
6. Flask Web App Development
7. Cloud Deployment on Render

## 💻 How to Run Locally

1. Clone the repository

```
git clone https://github.com/Rushikesh1232/student-performance-ml-project.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Flask application

```
python app/app.py
```

4. Open browser

```
http://localhost:10000
```
